# WL Zoom
- Basic Zoom Mod For WorldLoader
- Allows dynamic zooming
- Max Fov of 140
- Min Fov of 5
# Keybinds
- Zoom Camera   =   Mouse5 (Top Side Mouse)
- Reset Zoom   =   Middle Mouse
- Zoom In / Out   =   Scroll Wheel
